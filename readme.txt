# Burnout Detection API 🔥

AI-powered API for detecting employee burnout risk using XGBoost.

## API Endpoints

### POST `/predict`
Predicts burnout level based on:
- `designation` (1-5): Job level
- `resource_allocation` (1-10): Workload
- `mental_fatigue_score` (4-20): Mental health score

### GET `/health`
Check API status
